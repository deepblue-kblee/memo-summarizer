## 🚨 **REQUIRED: 작업 시작 전 필수 체크리스트**

**⚠️ CRITICAL**: 어떤 작업을 하기 전에 반드시 아래 단계를 순서대로 수행하세요.

### 📋 **필수 읽기 순서 (MUST READ)**

1. **📈 [PROGRESS.md](PROGRESS.md)** **← 현재 상태 파악**
   - **완료된 작업들** 및 현재 프로젝트 상태
   - **Git 상태**, 최근 변경사항, 파일 구조
   - **작업 연속성**: 이전 세션에서 어디까지 진행되었는지 파악

2. **📋 [PLAN.md](PLAN.md)** **← 다음 할 일 확인**
   - **우선순위별 작업 계획** (Priority 1, 2, 3 순서)
   - **진행 중인 작업** 및 즉시 진행 가능한 작업
   - **새 세션 시작 가이드** 및 체크리스트

3. **📚 [SYSTEM.md](SYSTEM.md)** (아키텍처 이해 필요시)
   - 전체 시스템 아키텍처 및 PARA 분류 규칙
   - 확장 및 개발 가이드

### ✅ **시작 체크리스트**
- [ ] **PROGRESS.md 읽고 현재 완료 상태 파악**
- [ ] **PLAN.md 읽고 다음 우선순위 작업 확인** (Priority 1, 2, 3 순서)
- [ ] **Git 상태 확인**: `git status && git log --oneline -3`
- [ ] **현재 브랜치 확인** 및 커밋 상태 파악
- [ ] **진행 중인 작업이 있는지 확인** (🚧 상태)

## ⚡ **빠른 상태 파악 명령어**

작업 시작 전 아래 명령어로 현재 상태를 빠르게 확인:

```bash
# 1. 현재 우선순위 작업 확인
grep -A 10 "Priority 1" .ai-docs/PLAN.md

# 2. 진행 중인 작업 확인
grep -A 5 "🚧 진행 중" .ai-docs/PLAN.md

# 3. Git 상태 빠른 확인
git log --oneline -3 && git status --short

# 4. 완료된 작업 확인 (중복 방지)
grep -A 10 "✅ 완료된 작업" .ai-docs/PROGRESS.md
```

```
    ↓

```

## 🚨 **중요한 제약 조건**

### **작업 우선순위 준수**
- **Priority 1**: 가장 중요한 작업 (🚨 마크)
- **Priority 2**: 문서 정리 등 (✅ 완료 또는 🚧 진행 중)
- **Priority 3**: 기능 확장 (장기 계획)

### **중복 작업 방지**
- **✅ 완료된 작업**: 다시 하지 말 것
- **🚧 진행 중인 작업**: 상태 확인 후 이어서 진행
- **새로운 작업**: PROJECT_STATUS.md에 추가 후 시작

### **Git 관리**
- **커밋 전**: 반드시 현재 상태 백업
- **브랜치 확인**: main 브랜치 상태 파악
- **변경사항**: 의미 있는 단위로 커밋

## 💡 **작업 연속성 보장**

### **이전 작업 컨텍스트 파악**
1. **최근 커밋 메시지** 확인으로 마지막 작업 내용 파악
2. **PROGRESS.md**의 "완료된 작업들" 섹션으로 기존 성과 확인
3. **PLAN.md**의 "🚧 진행 중인 작업" 섹션에서 미완료 작업 확인

### **Multi-AI 환경 고려**
- **이전 AI**: Claude 또는 Gemini가 이전에 작업했을 수 있음
- **작업 기록**: PROGRESS.md에서 어떤 AI가 무엇을 했는지 확인
- **설정 차이**: AI별 특화 설정 고려 (CLAUDE.md vs GEMINI.md)

## 🔥 **WARNING - 이 지침을 무시하면:**

- ❌ **중복 작업**: 이미 완료된 작업을 다시 수행
- ❌ **충돌 위험**: 진행 중인 작업과 겹쳐서 문제 발생
- ❌ **컨텍스트 손실**: 이전 작업 의도와 연결성 상실
- ❌ **우선순위 혼란**: 중요하지 않은 작업에 시간 낭비

## ✅ **이 지침을 따르면:**

- ✅ **효율적 작업**: 현재 가장 중요한 작업에 집중
- ✅ **작업 연속성**: 이전 세션과 매끄럽게 연결
- ✅ **중복 방지**: 이미 완료된 작업 재수행 방지
- ✅ **Multi-AI 협업**: 다른 AI와의 협업 최적화
